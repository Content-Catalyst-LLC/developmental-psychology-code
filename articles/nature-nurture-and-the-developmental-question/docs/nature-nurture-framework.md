# Nature-Nurture Developmental Framework

## Core constructs

| Construct | Developmental meaning |
|---|---|
| Biological sensitivity | Latent sensitivity or reactivity, not a real genotype or diagnosis |
| Structural risk | Unequal social, economic, institutional, and environmental exposure |
| Chronic adversity | Persistent adversity load |
| Caregiver support | Time-varying relational and caregiving support |
| Acute stress | Time-varying stress exposure |
| Institutional support | School or institutional developmental support |
| Disability support | Accessibility, accommodation, and support for developmental variation |
| Resource stability | Material and institutional stability |
| Intervention exposure | Later support or program exposure |
| Protective context | Composite support, accommodation, stability, and intervention |
| Development score | Synthetic developmental outcome |
| Sensitivity profile | Grouping by sensitivity and protective context |

## Interpretation

This workflow does not infer real genetics, diagnosis, risk, disability status, or developmental destiny. It models how biological sensitivity and environmental context can interact in synthetic longitudinal data.
