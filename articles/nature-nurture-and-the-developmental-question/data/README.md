# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_nature_nurture_panel.py
```

Primary generated files:

- `nature_nurture_development_panel.csv`
- `school_context_metadata.csv`
- `nature_nurture_child_profiles.csv`

The generated data contain no real child, family, school, clinical, genetic, disability, mental-health, or personal information and should not be interpreted as empirical evidence.
