# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_nature_nurture_panel.py` creates the synthetic nature-nurture developmental panel.
- `model_nature_nurture_development.py` estimates dynamic developmental and differential-susceptibility models.
- `nature_nurture_scenario_analysis.py` compares caregiver-support, institutional-support, disability-support, resource-stability, stress-reduction, intervention, structural-risk-reduction, and combined-protective-context scenarios.

Run:

```bash
python3 python/generate_nature_nurture_panel.py
python3 python/model_nature_nurture_development.py
python3 python/nature_nurture_scenario_analysis.py
```
