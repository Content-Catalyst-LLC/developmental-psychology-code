# C++

The C++ utility provides a simple CSV summary for development score, protective context, caregiver support, acute stress, biological sensitivity, and structural risk.

Run:

```bash
g++ -std=c++17 cpp/nature_nurture_summary.cpp -o outputs/nature_nurture_summary_cpp
./outputs/nature_nurture_summary_cpp data/nature_nurture_development_panel.csv
```
