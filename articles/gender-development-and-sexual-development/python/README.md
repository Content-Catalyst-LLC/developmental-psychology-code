# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_gender_sexual_development_panel.py` creates the synthetic adolescent-development panel.
- `model_gender_sexual_development.py` estimates adjustment, protective-context, and stigma models.
- `gender_sexual_development_scenario_analysis.py` compares family-support, social-recognition, consent-education, school-connectedness, anti-harassment, stigma-reduction, and combined protective-context scenarios.

Run:

```bash
python3 python/generate_gender_sexual_development_panel.py
python3 python/model_gender_sexual_development.py
python3 python/gender_sexual_development_scenario_analysis.py
```
