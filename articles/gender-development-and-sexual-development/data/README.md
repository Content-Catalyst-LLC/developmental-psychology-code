# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_gender_sexual_development_panel.py
```

Primary generated files:

- `gender_sexual_development_panel.csv`
- `school_context_metadata.csv`
- `adolescent_development_profiles.csv`

The generated data contain no real child, adolescent, family, school, clinical, identity, sexual orientation, sexual behavior, discipline, or personal information and should not be interpreted as empirical evidence.
