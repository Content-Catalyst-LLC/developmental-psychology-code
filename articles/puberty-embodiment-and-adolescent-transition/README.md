# Puberty, Embodiment, and Adolescent Transition

Research-grade companion directory for **Puberty, Embodiment, and Adolescent Transition**.

GitHub folder:  
https://github.com/Content-Catalyst-LLC/developmental-psychology-code/tree/main/articles/puberty-embodiment-and-adolescent-transition

## Purpose

This folder models puberty as a biological, embodied, social, and institutional developmental transition. The synthetic workflows examine pubertal progression, relative timing, family support, peer comparison, body concern, stigma, digital visibility, school support, health education, privacy protection, menstrual support, disability accommodation, anti-harassment climate, protective context, and adolescent adjustment.

## Ethical boundary

This is not a medical, diagnostic, mental-health, puberty-classification, gender/sexuality-classification, disability-classification, school-placement, discipline, body-ranking, or adolescent-surveillance tool. All data are synthetic.

## Quick start

```bash
cd articles/puberty-embodiment-and-adolescent-transition
make python
make scenarios
```
