# Notebooks

This folder is reserved for exploratory notebooks. The reproducible workflows live in `python/` and `r/`.

Suggested notebook sequence:

1. Load `data/cultural_development_panel.csv`.
2. Inspect family orientation, institutional fit, mismatch, support, and society-level variables.
3. Plot average developmental trajectories.
4. Compare cultural condition profiles.
5. Explore mismatch and support together.
6. Test alternative society-level assumptions.
7. Add migration-sensitive or language-support scenarios.
