# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_wisdom_meaning_panel.py` creates the synthetic later-life panel.
- `model_wisdom_meaning.py` estimates meaning and wisdom-index models.
- `wisdom_meaning_scenario_analysis.py` compares connection, reflection, adaptive support, legacy, dignity, service, community, and health-burden-reduction scenarios.

Run:

```bash
python3 python/generate_wisdom_meaning_panel.py
python3 python/model_wisdom_meaning.py
python3 python/wisdom_meaning_scenario_analysis.py
```
