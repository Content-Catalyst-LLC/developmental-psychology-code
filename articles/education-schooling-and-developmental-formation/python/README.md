# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_schooling_development_panel.py` creates the synthetic panel.
- `model_schooling_development.py` estimates development and connectedness models.
- `schooling_scenario_analysis.py` compares teacher support, peer belonging, curriculum opportunity, and stress-reduction scenarios.

Run:

```bash
python3 python/generate_schooling_development_panel.py
python3 python/model_schooling_development.py
python3 python/schooling_scenario_analysis.py
```
