# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_schooling_development_panel.py
```

Primary generated files:

- `schooling_development_panel.csv`
- `school_metadata.csv`
- `student_school_profiles.csv`

The generated data contain no real student information and should not be interpreted as empirical evidence.
