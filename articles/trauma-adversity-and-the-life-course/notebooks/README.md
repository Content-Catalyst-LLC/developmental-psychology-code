# Notebooks

This folder is reserved for exploratory notebooks. The reproducible workflows live in `python/` and `r/`.

Suggested notebook sequence:

1. Load `data/trauma_life_course_panel.csv`.
2. Inspect adversity, support, stability, and context distributions.
3. Plot adaptation trajectories.
4. Compare adversity-support profiles.
5. Explore cumulative adversity and timing weights.
6. Test alternative support and service-access assumptions.
7. Add intervention timing or trauma-informed system scenarios.
