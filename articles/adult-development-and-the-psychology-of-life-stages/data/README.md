# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_adult_development_panel.py
```

Primary generated files:

- `adult_development_life_stages_panel.csv`
- `context_metadata.csv`
- `person_adult_development_profiles.csv`

The generated data contain no real health, workplace, clinical, family, insurance, benefits, caregiver, or personal information and should not be interpreted as empirical evidence.
