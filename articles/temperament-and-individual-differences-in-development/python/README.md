# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_temperament_panel.py` creates the synthetic temperament and goodness-of-fit panel.
- `model_temperament_goodness_of_fit.py` estimates adjustment and goodness-of-fit models.
- `temperament_scenario_analysis.py` compares caregiver-support, classroom-fit, teacher-responsiveness, movement-flexibility, stress-reduction, accommodation, and combined support scenarios.

Run:

```bash
python3 python/generate_temperament_panel.py
python3 python/model_temperament_goodness_of_fit.py
python3 python/temperament_scenario_analysis.py
```
