# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_temperament_panel.py
```

Primary generated files:

- `temperament_individual_differences_panel.csv`
- `classroom_metadata.csv`
- `child_temperament_profiles.csv`

The generated data contain no real child, family, school, disability, clinical, mental-health, disciplinary, or personal information and should not be interpreted as empirical evidence.
