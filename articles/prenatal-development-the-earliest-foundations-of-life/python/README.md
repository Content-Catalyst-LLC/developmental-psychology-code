# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_prenatal_development_panel.py` creates the synthetic prenatal development panel.
- `model_prenatal_development.py` estimates early-outcome, stress, care, and profile models.
- `prenatal_scenario_analysis.py` compares prenatal-care, healthcare-access, nutrition, social-support, stress-reduction, environmental-protection, and combined public-health scenarios.

Run:

```bash
python3 python/generate_prenatal_development_panel.py
python3 python/model_prenatal_development.py
python3 python/prenatal_scenario_analysis.py
```
