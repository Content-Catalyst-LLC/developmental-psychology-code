# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_prenatal_development_panel.py
```

Primary generated files:

- `prenatal_development_foundations_panel.csv`
- `neighborhood_context_metadata.csv`
- `prenatal_case_profiles.csv`

The generated data contain no real pregnancy, maternal, fetal, infant, clinical, insurance, benefits, reproductive, neighborhood, or personal information and should not be interpreted as empirical evidence.
