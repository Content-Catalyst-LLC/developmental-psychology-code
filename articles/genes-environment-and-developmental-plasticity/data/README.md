# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_genes_environment_panel.py
```

Primary generated files:

- `genes_environment_plasticity_panel.csv`
- `context_metadata.csv`
- `child_plasticity_profiles.csv`

The generated data contain no real genetic, child, family, school, or clinical information and should not be interpreted as empirical evidence.
