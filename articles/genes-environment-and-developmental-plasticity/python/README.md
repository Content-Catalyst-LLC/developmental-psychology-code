# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_genes_environment_panel.py` creates the synthetic panel.
- `model_genes_environment_plasticity.py` estimates development and embedding models.
- `plasticity_scenario_analysis.py` compares care, nutrition, school support, and stress-reduction scenarios.

Run:

```bash
python3 python/generate_genes_environment_panel.py
python3 python/model_genes_environment_plasticity.py
python3 python/plasticity_scenario_analysis.py
```
