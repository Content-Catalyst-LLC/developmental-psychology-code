# Genes, Environment, and Developmental Plasticity Framework

## Developmental variables

| Domain | Developmental relevance |
|---|---|
| Biological sensitivity | Susceptibility, reactivity, responsiveness, inherited variation |
| Caregiving quality | Regulation, safety, language, attachment, support |
| Environmental stress | Threat, instability, adversity, chronic burden |
| Nutritional support | Growth, brain development, energy, health |
| School support | Learning conditions, adult support, peer context, opportunity |
| Neighborhood safety | Stress exposure, mobility, safety, public environment |
| Service access | Prevention, care, disability support, family support |
| Timing weight | Sensitive-period or developmental timing assumptions |
| Embedded stress | Accumulated stress carried forward as developmental history |
| Embedded support | Accumulated support carried forward as developmental history |

## Interpretation

This workflow does not ask whether genes or environment "win." It asks how developmental outcomes change when biological sensitivity, environmental exposure, support, stress, timing, and accumulated embodiment interact.
