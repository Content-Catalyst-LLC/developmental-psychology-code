# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_continuity_discontinuity_panel.py` creates the synthetic developmental-change panel.
- `model_continuity_discontinuity.py` estimates continuous, nonlinear, threshold, logistic, and profile models.
- `developmental_turning_point_scenarios.py` compares support, stress reduction, rupture reduction, intervention, and combined protective turning-point scenarios.

Run:

```bash
python3 python/generate_continuity_discontinuity_panel.py
python3 python/model_continuity_discontinuity.py
python3 python/developmental_turning_point_scenarios.py
```
