# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_continuity_discontinuity_panel.py
```

Primary generated files:

- `continuity_discontinuity_panel.csv`
- `context_metadata.csv`
- `developmental_change_profiles.csv`

The generated data contain no real child, adolescent, adult, family, school, clinic, disability, developmental-screening, trauma, or personal information and should not be interpreted as empirical evidence.
