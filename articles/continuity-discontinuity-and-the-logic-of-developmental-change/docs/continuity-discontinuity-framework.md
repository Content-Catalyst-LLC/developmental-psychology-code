# Continuity-Discontinuity Framework

## Core constructs

| Construct | Developmental meaning |
|---|---|
| Baseline functioning | Initial observed developmental status |
| Growth rate | Continuous incremental change over time |
| Curvature | Nonlinear acceleration or deceleration |
| Threshold time | Wave or time point at which a visible transition occurs |
| Threshold sensitivity | Whether the synthetic trajectory includes a threshold-sensitive shift |
| Threshold on | Indicator that the threshold has been reached |
| Logistic transition | Smooth transition function around threshold time |
| Contextual support | Baseline support condition |
| Current support | Time-varying support |
| Chronic stress | Persistent developmental constraint |
| Institutional rupture | Context-produced developmental disruption |
| Intervention exposure | Context-produced support or turning point |
| School support | Context-level developmental support |
| Resource stability | Contextual stability and institutional resources |
| Transition readiness | Composite support/resource/intervention minus stress/rupture |
| Outcome score | Synthetic developmental outcome |

## Interpretation

This workflow does not claim that real development is universally continuous or discontinuous. It models how both shapes can be represented, tested, compared, and ethically interpreted.
