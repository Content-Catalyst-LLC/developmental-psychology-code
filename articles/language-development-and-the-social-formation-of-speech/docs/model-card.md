# Model Card

## Name

Synthetic language development and social formation trajectory model.

## Intended use

Research education, article companion materials, methods teaching, longitudinal analysis practice, language-development modeling, multilingual ecology modeling, and reproducible workflow development.

## Inputs

Synthetic variables representing baseline language, responsive interaction, shared reading, joint attention, conversational turns, hearing support, multilingual exposure, chronic stress, language ecology support, book access, early education quality, home-language recognition, current interaction, current reading, current joint attention, current turn-taking, current stress, language support context, and language score.

## Outputs

Longitudinal synthetic panel, profile summaries, context summaries, dynamic regression outputs, nonlinear growth outputs, trajectory plots, scenario comparisons, and validation summaries.

## Data notes

All generated data are synthetic and parameterized for transparency. The design is suitable for demonstration and extension.

## Limitations

The workflow uses simplified mechanisms, simulated parameters, and no validated expressive-language, receptive-language, vocabulary, grammar, hearing, AAC, multilingual competence, pragmatic-language, literacy, or school-language measures.

## Extension paths

The structure can be extended with validated language assessments, hearing screenings, speech-language evaluations, classroom observations, caregiver reports, AAC access variables, multilingual exposure logs, school-language data, literacy measures, and stronger causal designs.
