# Notebooks

This folder is reserved for exploratory notebooks. The reproducible workflows live in `python/` and `r/`.

Suggested notebook sequence:

1. Load `data/developmental_timing_panel.csv`.
2. Plot the critical-period and sensitive-period weights.
3. Compare critical, sensitive, multi-window, and recovery outcomes.
4. Inspect experience-by-window interactions.
5. Test alternative timing-window assumptions.
6. Discuss ethical risks of deterministic interpretation.
