# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_aging_adaptation_panel.py` creates the synthetic later-life panel.
- `model_aging_adaptation.py` estimates adjustment and functional-fit models.
- `aging_adaptation_scenario_analysis.py` compares social support, adaptive strategy, accessibility, dignity-service, meaning, health-burden reduction, and combined support scenarios.

Run:

```bash
python3 python/generate_aging_adaptation_panel.py
python3 python/model_aging_adaptation.py
python3 python/aging_adaptation_scenario_analysis.py
```
