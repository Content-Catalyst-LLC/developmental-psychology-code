# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_aging_adaptation_panel.py
```

Primary generated files:

- `aging_adaptation_later_life_panel.csv`
- `care_context_metadata.csv`
- `person_aging_adaptation_profiles.csv`

The generated data contain no real health, cognitive, clinical, elder-care, fall-risk, insurance, benefits, caregiver, or personal information and should not be interpreted as empirical evidence.
