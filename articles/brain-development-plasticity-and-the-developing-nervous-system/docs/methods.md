# Methods

This workflow creates a synthetic longitudinal panel for brain development, plasticity, and developmental-outcome research. It demonstrates how neural state and developmental functioning can be represented across repeated waves while incorporating family support, learning opportunity, sleep, sensory regulation, health access, environmental risk, school support, neighborhood safety, acute stress, chronic stress, and developmental ecology.

The design includes child-level baseline variables, repeated waves, context-level support/risk variables, time-varying developmental inputs, neural state, developmental outcome, neurodevelopmental profiles, dynamic regression, scenario comparison, SQL summaries, and compiled-language validation utilities.
