# Responsible Use

This repository provides transparent synthetic workflows for neurodevelopmental trajectory modeling. Any empirical extension should include data provenance, child research protections, caregiver consent, disability-rights review, health-data governance, environmental-justice review, anti-bias review, and clear separation between descriptive modeling and causal claims.

Neurodevelopmental data should be used to understand systems of support, stress, safety, access, environmental exposure, sleep, health, learning, and developmental opportunity rather than to reduce children, disabilities, neurodivergent profiles, families, or communities to fixed hierarchies of brain-based ability.
