# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_family_systems_panel.py` creates the synthetic panel.
- `model_family_systems.py` estimates development and family-support models.
- `family_support_scenario_analysis.py` compares caregiver support, kin support, household stability, and stress-reduction scenarios.

Run:

```bash
python3 python/generate_family_systems_panel.py
python3 python/model_family_systems.py
python3 python/family_support_scenario_analysis.py
```
