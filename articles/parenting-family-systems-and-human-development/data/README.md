# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_family_systems_panel.py
```

Primary generated files:

- `family_systems_panel.csv`
- `household_metadata.csv`
- `child_family_profiles.csv`

The generated data contain no real child or family information and should not be interpreted as empirical evidence.
