# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_developmental_systems_panel.py
```

Primary generated files:

- `developmental_systems_panel.csv`
- `school_metadata.csv`
- `neighborhood_metadata.csv`
- `child_systems_profiles.csv`

The generated data contain no real child, family, school, neighborhood, genetic, clinical, or administrative information and should not be interpreted as empirical evidence.
