# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_lifespan_baltes_panel.py
```

Primary generated files:

- `lifespan_baltes_panel.csv`
- `cohort_metadata.csv`
- `person_lifespan_profiles.csv`

The generated data contain no real health, cognitive, clinical, aging, insurance, employment, or personal information and should not be interpreted as empirical evidence.
