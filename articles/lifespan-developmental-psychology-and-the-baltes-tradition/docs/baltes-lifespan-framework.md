# Baltes Lifespan Development Framework

## Core propositions

| Proposition | Developmental meaning |
|---|---|
| Lifelong development | Development extends from conception to death |
| Multidimensionality | Physical, cognitive, emotional, social, and motivational domains interact |
| Multidirectionality | Some capacities gain while others decline or reorganize |
| Plasticity | Change remains possible under supportive conditions, within limits |
| Historical embeddedness | Lives unfold within cohorts, institutions, technologies, policies, and cultural history |
| Gains and losses | Development always includes tradeoffs |
| Contextualism | Development reflects age-graded, history-graded, and nonnormative influences |
| SOC adaptation | People select goals, optimize resources, and compensate for loss |

## Interpretation

This workflow does not treat aging as simple decline or development as simple childhood growth. It models development as lifelong adaptation under changing resources and constraints.
