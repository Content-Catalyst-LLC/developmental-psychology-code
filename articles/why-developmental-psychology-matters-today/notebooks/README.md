# Notebooks

This folder is reserved for exploratory notebooks. To keep the repository lightweight and reviewable, the main reproducible workflows are stored as scripts in `python/` and `r/`.

Suggested notebook sequence:

1. Load `data/developmental_panel.csv`.
2. Inspect missingness and distributions.
3. Plot developmental trajectories.
4. Compare high-support and high-risk groups.
5. Reproduce the Python and R model summaries.
6. Add sensitivity checks for alternative model specifications.
