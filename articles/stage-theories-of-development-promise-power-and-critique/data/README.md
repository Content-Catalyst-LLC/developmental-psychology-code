# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_stage_theory_panel.py
```

Primary generated files:

- `stage_theory_development_panel.csv`
- `context_metadata.csv`
- `stage_theory_child_profiles.csv`

The generated data contain no real child, adolescent, adult, family, school, clinic, disability, developmental-screening, or personal information and should not be interpreted as empirical evidence.
