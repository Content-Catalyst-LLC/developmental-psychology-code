# Stage Theory Framework

## Core constructs

| Construct | Developmental meaning |
|---|---|
| Baseline functioning | Initial observed developmental status |
| Growth rate | Continuous incremental change over time |
| Threshold time | Wave or time point at which a stage-like shift becomes visible |
| Stage pattern | Whether a person follows a synthetic stage-like or mostly continuous pattern |
| Threshold on | Indicator that the threshold has been reached |
| Logistic transition | Smooth transition function around the threshold |
| Family support | Individual-level support context |
| Current support | Time-varying support |
| Chronic stress | Persistent developmental constraint |
| School support | Context-level developmental support |
| Resource stability | Contextual stability and institutional resources |
| Transition readiness | Composite of support and resources minus stress |
| Outcome score | Synthetic developmental outcome |

## Interpretation

This workflow does not claim that real development follows universal stages. It models how stage-like transitions can be represented, tested, compared, and critiqued.
