# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_stage_theory_panel.py` creates the synthetic stage-theory developmental panel.
- `model_stage_like_development.py` estimates threshold, logistic-transition, and profile models.
- `stage_transition_scenario_analysis.py` compares family-support, school-support, resource-stability, stress-reduction, transition-readiness, and combined-support scenarios.

Run:

```bash
python3 python/generate_stage_theory_panel.py
python3 python/model_stage_like_development.py
python3 python/stage_transition_scenario_analysis.py
```
