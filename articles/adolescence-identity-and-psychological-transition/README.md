# Adolescence, Identity, and Psychological Transition

Research-grade companion directory for the article **Adolescence, Identity, and Psychological Transition**.

GitHub folder:  
https://github.com/Content-Catalyst-LLC/developmental-psychology-code/tree/main/articles/adolescence-identity-and-psychological-transition

## Purpose

This folder models adolescence as a psychological, relational, embodied, institutional, and future-oriented transition. The workflows use synthetic longitudinal data to examine identity coherence across repeated waves under peer support, family support, school connectedness, future orientation, school climate, counseling access, extracurricular access, identity safety, digital safety, exclusion, and digital stress.

## Ethical boundary

This is not a clinical, mental-health, school-placement, maturity-scoring, discipline, identity-risk, gender/sexuality classification, disability-classification, or student-surveillance tool. All data are synthetic.

## Quick start

```bash
cd articles/adolescence-identity-and-psychological-transition
make python
make scenarios
```
