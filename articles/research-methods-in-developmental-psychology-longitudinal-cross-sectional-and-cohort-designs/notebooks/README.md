This folder is reserved for exploratory notebooks. The reproducible workflows live in `python/` and `r/`.

1. Load `data/developmental_design_panel.csv`.
2. Compare cross-sectional and longitudinal estimates.
3. Plot cohort-sequential trajectories.

5. Explore age-period-cohort confounding.
6. Test alternative design assumptions.
