# Python Workflow

This folder contains the main reproducible workflow.

## Scripts

- `generate_disability_neurodivergence_panel.py` creates the synthetic panel.
- `model_disability_neurodivergence.py` estimates development and participation models.
- `access_scenario_analysis.py` compares universal design, communication access, and barrier-reduction scenarios.

Run:

```bash
python3 python/generate_disability_neurodivergence_panel.py
python3 python/model_disability_neurodivergence.py
python3 python/access_scenario_analysis.py
```
