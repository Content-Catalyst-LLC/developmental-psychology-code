# Data

This folder contains the data dictionary and generated synthetic datasets.

Generate the main synthetic panel:

```bash
python3 ../python/generate_disability_neurodivergence_panel.py
```

Primary generated files:

- `disability_neurodivergence_panel.csv`
- `setting_metadata.csv`
- `child_access_profiles.csv`

The generated data contain no real personal information and should not be interpreted as empirical evidence.
