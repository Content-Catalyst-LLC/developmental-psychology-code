# Developmental Psychology Code

Open computational scaffolding for the Developmental Psychology knowledge series.

This repository supports article-level examples for lifespan development, prenatal development, temperament, brain development, cognitive development, language development, attachment, play, self-regulation, moral development, adolescence, adulthood, aging, developmental systems theory, family systems, education, disability, neurodivergence, developmental psychopathology, trauma, resilience, culture, inequality, longitudinal methods, and sensitive periods.

## Structure

Each article folder contains:

- `python/`
- `r/`
- `julia/`
- `cpp/`
- `fortran/`
- `c/`
- `rust/`
- `go/`
- `sql/`
- `notebooks/`
- `docs/`
- `data/raw/`
- `data/processed/`
- `outputs/`
